About LANdialler

LANdialler enables several computers on a LAN (Local Area Network) to share control of a dial up connection to the Internet.

If you have a small home or office network (a LAN) that connects to the Internet via a dial up link on a Linux computer, then LANdialler could be of real use to you. If you haven't got a Linux router yet find out how to set one up.

The LANdialler client enables any computer on your network to start or stop the Internet connection, via a simple and straightforward user interface.

The dial up connections are handled by your Linux router and your desktop computers remain safely protected from prying eyes by your router's firewall (assuming you have configured it - if not why not?).

LANdialler is released under the terms of the GNU Public License.

How does it work?

LANdialler consists of two programs.

  • The client runs on all desktop computers that require Internet access. It talks to the server, requesting Internet connectivity for as long as the user requires it.

  • The server runs on the Linux router and manages the state of the dial up connection. It keeps track of the number of people who have asked to go online and keeps the Internet connection up until the last user disconnects.

The client and server communicate with one another over the local network through the XML-RPC protocol.

Requirements

Both the client and server are written in Python, so you will need to make sure that you have a recent version installed.

The client should run on any operating system with Python and Tk support - it has been tested on both Windows and Linux.

The server should run on any POSIX compliant operating system, but has only been tested on Debian GNU/Linux v2.2. You could use any of the BSD variants (e.g. FreeBSD, OpenBSD, NetBSD), Solaris, etc. instead if you prefer. Just make sure you can configure it to route IP packets between your network and the Internet. There is lots of documentation available for Linux if you're not sure how to set it up.

Download

The latest versions of LANdialler are always available from the LANdialler project page. You must download the latest versions of all three packages:

  • landialler - the client
  • landiallerd - the server
  • gmalib - supporting library

Installation instructions are included with the source code.

You will also need to get the xmlrpclib library from PythonWare.
